# GrowthLog versioning
v1 represents the core functionality of the GrowthLog REST API. I am reserving v2 for the version in which I will integrate a ranking algorithm. v3 is the final version which I have in mind, with more advanced analysis of user interactions and with provisions for advertisers. In the API docs, I will only write what was added in the version under that version's folder. For example, I will only say what I added to the tech stack for v2, and will not write about the parts of the tech stack that I pre-existing from v1. Read the docs in order please.
